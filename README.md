# Resident-Evil-9-Requiem-Wiki
Flask og MariaDB-basert wiki-nettsted for Resident Evil 9. Målet med prosjektet er å demonstrere ferdigheter fra IT-programfagene ved å utvikle en webapplikasjon som kjører på en serverinfrastruktur og er tilgjengelig via internett.

Prosjektet er bygget ved hjelp av:

- Python
- Flask (webserver)
- MariaDB (database)
- HTML / CSS

---

# Prosjektmål
Målet med dette prosjektet er å demonstrere følgende ferdigheter:

- Programmering med Python
- Oppsett av serverinfrastruktur
- Bruk av en database for å lagre og hente data
- Dokumentasjon og versjonskontroll ved hjelp av GitHub
- Lage brukeropplæringsmateriell
- Testing og forbedring av systemet

---

# Funksjoner (planlagt)
Wikien vil inneholde informasjon om Resident Evil 9, som for eksempel:

- Karakterer
- Fiender
- Våpen
- Spillinformasjon

Systemet vil tillate CRUD-operasjoner:

Opprett – Legg til ny informasjon
Les – Vis informasjon
Oppdater – Rediger eksisterende informasjon
Slett – Fjern informasjon
